# Borgo | Domótica y Automatización de Ultra Lujo

Un portal digital interactivo de nivel premium diseñado y construido para **Borgo**, líder en integración de sistemas domóticos sofisticados y tecnología invisible para residencias exclusivas de alto standing.

Este proyecto ha sido optimizado con un diseño elegante, componentes animados con fluidez cinematográfica, tipografía sofisticada de alta legibilidad, soporte SEO profesional y un diseño adaptable (Mobile-First) optimizado para todos los dispositivos modernos.

---

## ✨ Características Principales

- **Diseño Ultra Premium (Framer Style)**: Inspirado en interfaces modernas y elegantes, utilizando paletas de colores sobrias y acentos cálidos con la tipografía serif elegante y mono de alta precisión.
- **Header Flotante Inteligente**: Un menú de navegación interactivo que se oculta inteligentemente al hacer scroll hacia abajo y reaparece elegantemente al subir o encontrarse en el inicio, garantizando una visualización libre de distracciones.
- **Sección FAQs Inteligente**: Menú interactivo y auto-ajustable con altura fija y desplazamiento fluido para evitar saltos o empujes visuales en el layout inferior del sitio.
- **Formulario de Contacto Premium**: Completa experiencia de agendamiento y envío de mensajes integrados, totalmente reactivos.
- **SEO Profesional & Open Graph**: Soporte avanzado para indexación en motores de búsqueda, con etiquetas meta, Twitter Cards y Open Graph optimizados.
- **Rendimiento Óptimo**: Desarrollado con **React 18**, **TypeScript**, **Tailwind CSS** y animaciones fluidas impulsadas por **Framer Motion** (`motion`).

## 🚀 Inicio Rápido

### Requisitos Previos

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 18 o superior) en tu entorno local.

### Instalación

1. Clona el proyecto o descarga el archivo fuente.
2. Abre la terminal en el directorio raíz del proyecto y ejecuta:
   ```bash
   npm install
   ```

### Servidor de Desarrollo

Inicia el entorno local interactivo ejecutando:
```bash
npm run dev
```
El servidor se abrirá de forma predeterminada en `http://localhost:3000`.

### Compilación para Producción

Genera el paquete optimizado y comprimido listo para subir a producción ejecutando:
```bash
npm run build
```
Los archivos compilados listos para servir se encontrarán dentro de la carpeta `/dist`.
